# Character Images

Place your 3D character illustration images here.

Expected filenames:
- `ai-character.png`        → AI-Powered Solutions
- `consulting-character.png` → IT Consulting Solution
- `security-character.png`  → Cyber Security Solutions
- `cloud-character.png`     → Cloud Infra Solutions
- `mobile-character.png`    → Mobile App Developments
- `ecommerce-character.png` → eCommerce Solutions

If an image is missing, the card will display a coloured icon placeholder automatically.
